# Учебный проект

Проект для выполнения заданий учебного курса.

## Структура

- `tasks/` - отдельные задания курса.
- `tasks/task_01/` - первое задание и шаблон для следующих задач.
- `materials/` - материалы курса, условия, файлы от преподавателя и загруженные данные.
- `src/` - общий код, если он понадобится нескольким заданиям.
- `tests/` - общие тесты.

## Как добавлять новое задание

1. Создать папку `tasks/task_02`, `tasks/task_03` и так далее.
2. Добавить туда файл с решением, например `solution.py`.
3. Описать условие и заметки в `README.md` внутри папки задания.

## Запуск примера

```powershell
python tasks/task_01/solution.py
```

## API-ключ Gemini

Ключ Gemini хранится в локальном файле `.env`:

```text
GEMINI_API_KEY=your_gemini_api_key_here
```

Файл `.env` добавлен в `.gitignore`, поэтому настоящий ключ не должен попадать в git.
