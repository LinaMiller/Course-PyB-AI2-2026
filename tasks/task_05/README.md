# Task 5: Painting Classifier

## Goal

Classify artworks by creation year.

Paintings created before 1900 are classified as classic. Paintings created in
1900 or later are classified as modern.

The program also adds two new artworks to the original lists and checks that
the classification still works.

## New Classification Ideas

The museum can also classify paintings by:

- artistic style;
- dominant color;
- painting size;
- subject, such as portrait, landscape, or abstract art.

This solution implements classification by artistic style.

## Files

- `solution.py` - painting classification task.
- `apple_or_banana.py` - fruit classification task.
