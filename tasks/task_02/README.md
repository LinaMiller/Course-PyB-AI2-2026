# Task 2: Tokenizer

## Goal

Explore how an AI tokenizer splits human language into tokens and converts
those tokens into numeric IDs.

## Requirements

The program should:

- create a sentence variable;
- print the sentence length, including spaces;
- print the number of spaces;
- split the sentence into tokens;
- convert the tokens into token IDs;
- run a few small experiments with English and Hebrew text.

- `solution.py` - solution for the task.
