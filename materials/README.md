# Материалы курса

Здесь лежат исходные материалы курса: презентации, документы с заданиями,
PDF-файлы, примеры данных и другие файлы от преподавателя.

## Структура

- `task_00/` - вводные материалы курса.
- `task_01/` - материалы для первого проекта "Balanced Day".
- `task_02/` - материалы для второго задания.
- `task_03/` - материалы для третьего задания.
- `task_04/` - материалы для четвертого задания.

Для следующих заданий можно создавать такие же папки:

```text
materials/task_02/
materials/task_03/
materials/task_04/
```

Так материалы и решения будут идти параллельно:

```text
materials/task_01/  # условие, презентация, примеры
tasks/task_01/      # код решения
```
